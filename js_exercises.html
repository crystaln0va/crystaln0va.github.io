JavaScript Exercises

1. What is the difference between the following 2 statements?

setTimeout(booyah, 2000);
setTimeout(booyah(), 2000);

We need booyah to be a callback function, or a function who does not execute until a specific desired time. In this case it would be after 2000 milliseconds, or 2 seconds. If we place the parentheses after booyah as booyah(), the function will be called immediately, and this will defeat the purpose of setting the timer.

2. What do the following 2 alerts display (answer without running the code)?

var myfunc = function(a, x) {
return a * x;
};
var x = myfunc(2, 3);
var y = myfunc;
alert(x);
alert(y(2,3));

In the first case, this alert will simply display the result of calling the function myfunc. Two arguments are provided, and they will pass through the parameters a and x, respectively. They will be multiplied together, and the outcome will be 6.

In the second case, y stores the function myfunc. When the function is called, two arguments are passed, 2 and 3, respectively. Ultimately, this is only the same action as before, and the arguments will be accepted and multiplied together. Thus, the outcome in this case is also 6.

4. What is "Unobtrusive Javascript"? What is the practical application of Unobtrusive Javascript (and the reasons for using it)?

Unobtrusive JavaScript is the organization of JavaScript code into a separate script file. It is used to keep code clean, organized, accessible, and simple. It puts all of the JavaScript into one place, so that modifications are easy to implement, and the location of elements is easy and clear.
